Application favorite 3

Discord, un monde social à portée de mains.

Ce réseau social est le meilleur que j’ai pu trouver. Une
application simple d’utilisation qui permet de regrouper
un maximum de personnes sans difficulté grâce aux différents salons.
Personnellement je l’utilise pour retrouver mes amis dans différentes catégories.
Le mode sombre permet d’y rester jour et nuit
sans être pour autant fatigué. Le statut permet de vite repérer
avec qui je pourrais faire la prochaine partie de mon jeu du moment. De plus,
les émojis personnalisables développe la créativité ! Tant pour le personnel que
pour le professionnel je conseil Discord, voici le lien téléchargeable :
